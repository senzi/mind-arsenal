# 思维器阁 (Mind Arsenal)

一个基于 Vue 3 + Vite 开发的思维工具集合应用。

## 功能特点

- 七种思维工具，助你破局致远
- 优雅的用户界面，传统东方美学设计
- 支持自定义 API 设置
- 响应式布局，适配多种设备

## 技术栈

- Vue 3
- Vite
- Local Storage for settings
- RESTful API integration

## 开发环境设置

1. 安装依赖
```bash
npm install
```

2. 启动开发服务器
```bash
npm run dev
```

3. 构建生产版本
```bash
npm run build
```

## API 配置

在设置面板中配置：
- Base URL（默认：https://api.link-ai.tech/v1）
- API Key
